Herramientas

Soy el creador de las siguientes herramientas:

  • Zunlocker (Herramienta rápida para descifrar una contraseña de un archivo zip).
  • WiFiDecryptor (Herramienta automática y rápida para obtener la contraseña de una red WiFi).
  • HDecrypt (Herramienta rápida que descifra hashes. Algoritmos hash soportados: [MD5, SHA1, SHA224, SHA226, SHA384 SHA512]).
  • Xrpck (Herramienta que explota el recurso de WordPress “xmlrpc.php” para realizar un ataque de diccionario de fuerza bruta a un usuario para extraer su contraseña).
